<CardGrid stagger>
  <Card title="Safe by default" icon="warning">
    Always preview before moving. `ordr preview` shows exactly what would happen — no surprises.
  </Card>
  <Card title="Reversible" icon="left-arrow">
    Made a mistake? `ordr undo` reverses the last clean operation instantly.
  </Card>
  <Card title="Flexible rules" icon="setting">
    Match files by extension, name pattern, size, or age. Rules compose cleanly.
  </Card>
  <Card title="Homebrew native" icon="download">
    Single binary, zero runtime dependencies. Install once, works everywhere.
  </Card>
